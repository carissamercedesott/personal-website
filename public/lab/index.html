Experiments

The Lab

Small interaction experiments, fully working, built in TypeScript + Solid on this site's design tokens. Every demo works with a keyboard alone and respects reduced motion — and each one ships its own source, right underneath.

01

Command palette

Fuzzy search over this site's pages — ⌘K works anywhere on this page, and it really navigates.

02

Ghost-text autocomplete

Editor-style inline completions — the suggestion continues exactly where your caret is, Tab accepts.

03

Animated diff

A code block that transitions between before and after, one line at a time — removals fold away, additions slide in.

04

Streaming token renderer

Text that arrives word by word with a cursor and a mid-stream "thinking" pause — tune the speed and feel the character change.

05

Motion toy

This site's easing and duration tokens, made touchable — drag the card (or shove it with arrow keys) and compare curves against a real spring.

06

Ragdoll in a box

The site's resident smiski, off duty — the same rig that walks you through the homepage, here with six rigid bodies, five joints, and one box. Grab any limb and fling him; arrow keys shove, Enter tosses. (3D loads only when you scroll here.)
