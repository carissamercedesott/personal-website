The Playground

This is the space I use to design and experiment with new interactive motions and animations.

Just for funsies, I like making interactive components with micro-details that make the interface feel alive. Go ahead, try it out! Type, click, drag, throw. And if you like how something feels, its source code is right underneath :)

01

Command palette

The fastest way around any product is the keyboard — fuzzy search that meets your hands where they already are.

try it Press ⌘K anywhere on this page, type “design”, hit ↵ — it really navigates.

02

Ghost-text autocomplete

Editor-style inline completion — the interface finishes your sentence without ever stealing your caret.

try it Type “inter…” in the box and the rest appears in grey — Tab accepts, Esc dismisses.

03

Animated diff

Code change as motion — instead of two walls of red and green, the block plays the edit: removals fold away, additions slide in.

try it Click “Apply the diff”, watch the edit play, then run it backward.

04

Streaming token renderer

AI chat lives or dies by rhythm — the same words feel calm or frantic depending on how they arrive.

try it Press play, then drag the speed slider mid-stream and feel the character change.

05

Motion toy

This site's easing and duration tokens, made touchable — motion you compare by feel instead of by graph.

try it Drag the card and let go (arrow keys shove it too), then switch curves and feel the difference.

06

Ragdoll in a box

The site's resident smiski, off duty — the same rig that walks you through the homepage, here as six rigid bodies and five joints in a box. (3D loads only when you scroll here.)

try it Grab any limb and fling him — arrow keys shove, Enter tosses.
